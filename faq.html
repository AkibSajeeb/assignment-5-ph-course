Important FAQ..

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents that represents the structure of a webpage as a tree of objects, allowing developers to access, modify, and manipulate HTML and CSS elements using languages like JavaScript

How do you select the document from the DOM?

You can select the document from the DOM using document, which represents the entire webpage, allowing access to its elements and content for manipulation.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element to handle events from its child elements, rather than adding individual listeners to each child. It's useful because it improves performance, especially when dealing with many elements, and it allows handling dynamic content that may be added later.

How do you manipulates an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, I wil use getAttribute() and setAttribute() to retrieve and modify attributes, and manipulate styles directly with element.style.property or manage CSS classes using classList.add(), classList.remove(), and classList.toggle()
